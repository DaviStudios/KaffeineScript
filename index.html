KaffeineScript

A little, funny Esolang.

KEYWORDS <>

drop (variable name) = (string, integer, table etc.) - Variable Declaration

int (variable name) - change an variable to an integer

str (variable name) - change an variable to an string

mug (function name) = [functions here, put them in strings] - Function Declaration

sugar (function name) - Run an function

spoon (variable name) = (new value) - Change an variable

splash (string, variable etc) - output text

DOWNLOAD & USAGE <>

1. Click on the KS logo

2. Unzip it, you will see the ks.exe file.

3. Put ks.exe in any directory you want

4. create a file with any name you want, make sure it uses the .kaffe extensions

5. open the terminal and input in the instruction on num 5.

6. ks.exe (your file name)

It may say that its dangerous, just because its unknown :)
